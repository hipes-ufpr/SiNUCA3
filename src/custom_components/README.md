Any .c or .cpp file you add here will be picked up by the build system!

You may want to run `bear --append -- make` after adding new files so they're
added to the compilation database for tools such as ClangD.

Qualquer arquivo .c ou .cpp que você adicionar aqui será automaticamente
encontrado pelo sistema de build!

Você pode querer rodar `bear --append -- make` depois de adicionar arquivos
novos para que eles sejam adicionados ao banco de dados de compilação para
ferramentas como o ClangD.
